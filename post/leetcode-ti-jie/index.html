9.回文数

题目大意

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

解题思路

反转一半的数字。

代码

class Solution{

    public boolean isPalindrome(int x){
		#如果小于0，或者末尾为0，则直接返回false
        if(x<0||(x%10 == 0 &&x!=0)){
            return false;
        }
        int revertedNumber = 0;
		#当原始x小于或等于反转后的数字时，说明已经到中间循环结束。
        while(x>revertedNumber){
            revertedNumber = revertedNumber *10 + x%10;
            x /= 10;
        }
        return x == revertedNumber ||x==revertedNumber/10;
    }
}

13.罗马数字转整数

题目大意

给定一个罗马数字，将其转换成整数。

解题思路

从前往后读字符串，如果在前的字符大于或等于在后的字符，则相加，如果在前的字符小于在后的字符，则减去较小的字符。

代码

	java import java.util.*;
	class Solution {

    public int romanToInt(String s) {
        int sum = 0;
        int preNum = getValue(s.charAt(0));
        for(int i = 1;i < s.length(); i ++) {
            int num = getValue(s.charAt(i));
            if(preNum < num) {
                sum -= preNum;
            } else {
                sum += preNum;
            }
            preNum = num;
        }
        sum += preNum;
        return sum;
    }
    
    private int getValue(char ch) {
        switch(ch) {
            case 'I': return 1;
            case 'V': return 5;
            case 'X': return 10;
            case 'L': return 50;
            case 'C': return 100;
            case 'D': return 500;
            case 'M': return 1000;
            default: return 0;
        }
    }
}

20.有效的括号

题目大意

给定一个只包括括号的字符串，判断字符串是否有效。

  • 左括号必须用相同类型的右括号闭合。
  • 左括号必须以正确的顺序闭合。

解题思路

利用栈求解，如果是左括号就入栈，如果是右括号，则判断是否与栈顶的左括号匹配
边界条件：
-- 1.如果碰到右括号而栈为空，则直接返回。
-- 2.如果右括号全部匹配完毕而栈还不为空，则直接返回。

代码

class Solution{
    public boolean isValid(String s){
        LinkedList<Character> list = new LinkedList<>();
        char[] chars = s.toCharArray();
        for (int i = 0;i<chars.length;i++){
            char c = chars[i];
            if(isLeft(c)){
                //左括号，入栈
                list.push(c);
            }
            else{
                //右括号，上一个左括号与之不匹配，不用再判断，直接返回
                if(list.isEmpty()||!isMatch(list.pop(),c)){
                    return false;
                }
            }
        }
        return list.isEmpty();
    }
    private boolean isLeft(char c){
        if (c =='(')
        return true;
        else if (c=='[')
        return true;
        else if(c=='{')
        return true;
        else
        return false;
    }
    private boolean isMatch(char a ,char b){
        switch(a){
            case '(':
            return b ==')';
            case '[':
                return b == ']';
            case '{':
                return b == '}';
            default:
                return false;
        }
    }
}

21.合并两个有序链表

题目大意

将两个升序链表合并为一个新的升序链表并返回。
实例：

输入：1->2->4,1->3->4
输出：1->1->2->3->4->4

解题思路

递归思想，如果第一个链表的最小值小于第二个链表的最小值，则对第一个链表该值后面的链表和第二个链表做递归，反之亦然。
终止条件： 如果第一个链表为空，则返回第二个链表，反之亦然。

代码

class Solution{
    public ListNode mergeTwoLists(ListNode l1,ListNode l2){
        if (l1 == null){
            return l2;
        }
        if (l2 == null){
            return l1;
        }
        else if (l1.val < l2.val){
            l1.next = mergeTwoLists(l1.next,l2);
            return l1;
        }
        else{
            l2.next = mergeTwoLists(l1,l2.next);
            return l2;
        }
    }
}

26.删除排序数组中的重复项

题目大意

给定一个排序数组，你需要在原地删除重复出现的元素，使得每个元素只出现一次，返回移除后数组的新长度。
必须在原地修改输入数据并在使用O(1)额外空间的条件下完成。

解题思路

双指针，如果[j] 与[i] 不相等，则i往后移，并给nums[i] 赋值为nums[j]，直到 j 遍历完毕。

代码

class Solution {
    public int removeDuplicates(int[] nums) {
    if (nums.length == 0) return 0;
    int i = 0;
    for (int j = 1; j < nums.length; j++) {
        if (nums[j] != nums[i]) {
            i++;
            nums[i] = nums[j];
        }
    }
    return i + 1;
}
}
