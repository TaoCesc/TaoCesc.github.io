1.两数之和

题目大意

给定一个整数数组nums和一个整数目标值 target，请你在该数组中找出和为目标值 target 的两个整数

解题思路

用HashMap存储。

代码

class Solution {
    public int maxArea(int[] height) {
        int left = 0,right = height.length-1;
        int max = 0;
        while(left < right){
            int cur = (right - left)*Math.min(height[right],height[left]);
            max = Math.max(cur,max);
            if(height[left] > height[right]){
                right --;
            }
            else{
                left++;
            }
        }
        return max;
    }
}

21.合并两个有序链表

题目大意

将两个升序链表合并为一个新的升序链表并返回。
实例：

输入：1->2->4,1->3->4
输出：1->1->2->3->4->4

解题思路

递归思想，如果第一个链表的最小值小于第二个链表的最小值，则对第一个链表该值后面的链表和第二个链表做递归，反之亦然。
终止条件： 如果第一个链表为空，则返回第二个链表，反之亦然。

代码

class Solution{
    public ListNode mergeTwoLists(ListNode l1,ListNode l2){
        if (l1 == null){
            return l2;
        }
        if (l2 == null){
            return l1;
        }
        else if (l1.val < l2.val){
            l1.next = mergeTwoLists(l1.next,l2);
            return l1;
        }
        else{
            l2.next = mergeTwoLists(l1,l2.next);
            return l2;
        }
    }
}

26.删除排序数组中的重复项

题目大意

给定一个排序数组，你需要在原地删除重复出现的元素，使得每个元素只出现一次，返回移除后数组的新长度。
必须在原地修改输入数据并在使用O(1)额外空间的条件下完成。

解题思路

双指针，如果[j] 与[i] 不相等，则i往后移，并给nums[i] 赋值为nums[j]，直到 j 遍历完毕。

代码

class Solution {
    public int removeDuplicates(int[] nums) {
    if (nums.length == 0) return 0;
    int i = 0;
    for (int j = 1; j < nums.length; j++) {
        if (nums[j] != nums[i]) {
            i++;
            nums[i] = nums[j];
        }
    }
    return i + 1;
}
}
